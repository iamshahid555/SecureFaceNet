# Evaluation

This folder contains evaluation and visualization scripts used to analyze
the behavior of the SecureFaceNet system.

## Available Visualizations

### DP-SGD Training Loss

- `plot_dp_loss.py`
- Visualizes training loss over epochs during differentially private training
- Helps analyze convergence behavior under differential privacy constraints

These visualizations are intended for analysis and reporting purposes and
do not operate on raw facial images.
